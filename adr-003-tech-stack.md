# ADR-003: Технологічний стек Backend — Python + FastAPI
 
## Статус
Accepted
 
## Контекст
Потрібно створити API для веб-інтерфейсу диспетчера та мобільного додатка майстра. Необхідна швидка інтеграція з зовнішніми системами та автоматична генерація документації API.
 
## Рішення
Використовувати мову програмування Python та фреймворк FastAPI.
 
## Обґрунтування
1. **Автодокументація:** Автоматична генерація Swagger/OpenAPI специфікації "з коробки".
2. **Асинхронність:** Ефективна обробка запитів до зовнішніх систем (Billing, SMS-шлюз).
3. **Швидкість написання:** Лаконічність Python прискорює розробку бізнес-логіки.
 
## Альтернативи
- **Node.js:** Відхилено через менш зручну екосистему для технічних розрахунків.
- **Java Spring:** Відхилено через тривалий процес конфігурації.
 
## Наслідки
### Позитивні
- Готова документація API для розробників фронтенду.
- Висока продуктивність.
### Негативні
- Потребує суворого дотримання типізації через Pydantic.
